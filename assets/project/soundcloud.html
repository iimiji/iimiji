Soundcloud Redesign

Redesigning the Soundcloud Application Features. For this project, I work individually to design a new feature for an existing mobile application, so I must research the current interface, interaction patterns, features, and users. This research will help inform the development of my new feature for the existing application. 

Used Tools: Figma, Protopie

Soundcloud Feature Redesign

User Experience

I researched the user experience to see what new features users need. By integrating the opinions of the above five users, I was able to narrow the direction of improvement with two functions.

User Persona

For more detailed access, the user persona features Louis who listens to music in the sound cloud and sings along with his favorite rapper to become a rapper. However, Louis is unhappy with the app interface about having to find lyrics separately outside the app and having unclear repeat options.

User Flow

This is a large-frame user flow that users experience using the app. Among them, I will focus on Track and Search and discover new features.

LOW FIDELITY Prototype

To design new features, I considered our new design with Nielsen’s Rules and Heuristics, especially focused on 1. Help and Documentation 2. Recognition rather than recall 3. Aesthetic and minimalist design.

Designing Low Fidelity, I investigated how these functions were designed and used in other music applications. Through this process, I was able to find out the common problems and some solutions.

Color Palette

Before the functional design, I designed the necessary icons that match the existing Sound Cloud.

Feature 01

Two more features have been added to the right of the search bar to provide more search options. The first is the music search function. This music search function can be used to find the song when users like to find the music playing in a restaurant or cafe or to find a piece that users remember the lyrics and melody but do not remember the song title. As shown in the following screen, if the user clicks the start listening button in the place where music is playing, AI finds and displays a song. And the icon shows a moving animation when the feature look for a song while listening to it. The music users find will play immediately, and the user can share or follow the artist directly. All interactions are unified into taps to simplify the user's experience, and when the icon is clicked, the background colour becomes dark, and the icon is connected at a glance.

Feature 02

The second function is a search filter with card UI. The filter asks if the user wants to search for an artist, a track or an album and provides a search function accordingly. In addition, a desired genre of music may be preset. This is because there can be multiple genres of music with the same title or artist. / The last search options are the country and language filters, which had many options to choose from, so I decided to provide a different UI with this filter option to list all of the options in alphabetical order, and there are options to search through typing at the same time as well. This feature will be helpful for users who are looking for songs released in a specific country or in a specific language.
